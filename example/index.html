flow-view

is a visual editor for Dataflow programming, powered by React.

  • Double click and write a node name to create it.
  • Click on a node to select it.
  • Add or remove input and output pins
  • Drag from an output to an input to connect them.
  • Click on a link to select it.
  • Delete nodes and links.
  • Rename nodes.
